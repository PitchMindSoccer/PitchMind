## Key Coaching Insights Extracted

### Principle 1: Timing is Universal
Across all 12 topics, **timing of movement** is critical:
- Losing marker: timing when to move
- Overlapping: timing the run
- Breaking lines: timing the pass and run
- Switching play: timing when to switch

**Teaching point**: Coaches should emphasize synchronized movement between ball carrier and receiver.

### Principle 2: Decision-Making Precedes Execution
All topics emphasize **recognition before execution**:
- When to switch vs. when not to
- When to break lines vs. when to progress normally
- When to overlap vs. underlap
- Which combination to use under pressure

**Teaching point**: Elite players think 1-2 seconds ahead; developing players react to what's in front of them.

### Principle 3: Position Determines Context
Every topic has position-specific variations:
- Fullbacks overlap/underlap differently than midfielders
- Center-backs break lines differently than attacking midfielders
- Forwards create space for others; midfielders create for themselves

**Teaching point**: "Universal first, position-specific second" is the correct teaching hierarchy.

### Principle 4: Progression = Unopposed → Semi-Opposed → Game-Realistic
Consistent teaching structure across all Coaches' Voice programs:
1. **Unopposed**: Learn the technique without pressure
2. **Semi-opposed**: Learn under light/moderate pressure
3. **Game-realistic**: Apply in realistic match scenarios

**Teaching point**: This progression should be non-negotiable in every coaching session.
